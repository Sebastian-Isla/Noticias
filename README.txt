# Proyecto: Sitio de Noticias

## Descripción
Este proyecto es un sitio web de noticias desarrollado en **PHP Nativo** utilizando el motor de plantillas **Twig**. Permite almacenar y mostrar artículos de noticias desde una base de datos MySQL. Además, incluye seguimiento con **Google Analytics 4 (GA4)**.

## Características
- Gestión de artículos/noticias almacenados en una base de datos MySQL.
- Visualización de noticias con videos embebidos.
- Diseño sencillo con colores verde y blanco.
- Integración con **Google Analytics 4 (GA4)** para seguimiento de visitas.

---

## Requisitos
Para ejecutar este proyecto necesitas:
- **XAMPP** o cualquier servidor que soporte PHP (versión 7.4 o superior).
- **MySQL** para la base de datos.
- **Composer** para gestionar dependencias (Twig).
